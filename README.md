# Chess Engine

## Propósito
Este motor de ajedrez proporciona la lógica central para jugar partidas. Incluye la representación del tablero, reglas de movimiento y evaluaciones de estado.

## Búsqueda Alfa-Beta
El motor emplea un algoritmo de búsqueda alfa-beta para explorar posibles movimientos, podando ramas no prometedoras y mejorando la eficiencia en la toma de decisiones.

## Uso
1. Ejecuta la clase principal.
2. Realiza movimientos a través de las funciones expuestas.
3. La búsqueda se activa automáticamente al evaluarse la siguiente jugada.
